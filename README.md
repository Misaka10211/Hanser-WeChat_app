# Hanser-WeChat_app
附上参考GitHub地址：https://github.com/Silverados/We-AnswerPage

目前小程序实现的功能有：
1.支持tag选题
2.支持难度选题
3.自动判断对错
4.显示解析

期望在将来实现的功能：
1.报错/反馈按钮：发现答案有错时向开发者反馈
2.题目预览功能：类似目录，便于在直播活动时快速找到题目
3.搜索框查找功能

目前的不足有：
1.界面的美化不足
2.tag过滤时间过长，可能导致第一题加载过慢
3.代码的耦合度较高
